install packages

$ sudo apt -y install python3 python3-venv python3-pip

$ python3 -m venv venv
$ source venv/bin/activate

$ python3 -m pip install pygame

create symlink workaround (https://github.com/pygame/pygame/issues/3756)

$ sudo ln -s /usr/share/alsa /usr/local/share/alsa

view midi devices in pygame

$ python3

inside python

>>> import pygame.midi
>>> pygame.midi.init()
>>> pygame.midi.get_count()
